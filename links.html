Изучение HTML


В данном цикле руководств вы познакомитесь с HTML более детально. На самом деле HTML не так сложен в понимании, но для полноценной работы необходимо знать его основы, как он устроен и как он работает.


Стилизация HTML с помощью CSS


Здесь вам подробно расскажут как с помощь CSS стилизовать ваш Web контент, как применять к содержимому шрифты, стилизовать изображения, менять цвет чего угодно, ну и как делать раскладку вашего контента. Рекомендую не передвигаться в процессе обучения по левому меню навигации, т.к. некоторые пункты из модуля по какой-то причине там отсутствуют.
